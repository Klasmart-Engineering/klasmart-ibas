### Setup

- Clone this repository
- Go to api directory `/backend`
- Copy file `.env.example` to `.env`
- Run `yarn install` on terminal
- Run `yarn develop` on terminal
- Your API will running on `localhost:1337`

### Run Local Server

- Go to api directory `/backend`
- Run `yarn develop` on terminal
- Your API will running on `localhost:1337`
